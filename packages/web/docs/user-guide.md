# UnicodeArtJs Web 用户指南

## 快速开始

1. 打开网页
2. 选择工作模式：**图片转字符画** 或 **文字Banner**
3. 上传图片或输入文字
4. 调整参数
5. 查看预览结果
6. 导出或复制

## 功能详解

### 图片转字符画

- **上传**: 拖拽图片或点击上传区域
- **支持格式**: JPG/PNG/GIF/WebP
- **大小限制**: 最大10MB
- **参数**: 高度/宽度/字符集/字体/矩阵大小等

### 文字Banner

- **输入**: 在下方的文本框中输入文字
- **实时预览**: 输入后自动生成字符画
- **参数**: 高度/宽度/字符集/字体等

### 字符集

| 字符集 | 说明 |
|--------|------|
| ASCII | 英文 + 符号 (95个可打印字符) |
| 扩展字符 | 更多符号和装饰字符 |
| 简体中文 | GB2312常用汉字 |
| 自定义 | 手动输入字符（从暗到亮排列） |

### 字体

- **视觉字体**: 用于渲染输入文本（采样用）
- **字素字体**: 用于预览区显示字符画（等宽字体）

### 裱框

- **样式**: 单线/双线/圆角/粗线/经典/ASCII等
- **配置**: 内边距、外边距、标题、阴影

### 主题

- **默认**: 亮色简约
- **暗黑**: 深色护眼
- **高对比度**: 黑白分明
- **Solarized浅色**: 温暖柔和
- **Nord极冰**: 冷色调

### 导出

- **TXT**: 纯文本格式
- **HTML**: 带样式的网页
- **PNG**: 图片格式
- **复制**: 直接复制到剪贴板

## 键盘快捷键

- `Tab` 切换到下一个输入
- `Enter` 在文字模式中换行

## 常见问题

**Q**: 文字转字符画时提示"生成失败"
**A**: 请检查Core库是否正确加载（查看浏览器控制台是否报错）

**Q**: PNG导出效果不理想
**A**: PNG使用Canvas渲染，效果取决于浏览器字体支持

**Q**: 设置保存了吗？
**A**: 所有参数自动保存在浏览器本地存储中
